Instrucciones de Uso

Para registrar nueva avería:

  1. Completa todos los campos del formulario
  2. Los campos se convierten automáticamente a MAYÚSCULAS
  3. Haz clic en "Guardar" para registrar

Para buscar registros:

  1. Escribe en el campo de búsqueda para ver sugerencias
  2. Usa los filtros por marca o código
  3. Haz clic en cualquier registro para editarlo

Instalar como app en móvil:

  1. Abre en Chrome (Android) o Safari (iOS)
  2. Toca el menú (⋮ o compartir)
  3. Selecciona "Añadir a pantalla de inicio"
  4. Sigue las instrucciones para instalar
